# Pokemon-Rubis-Guide

Je fait un super guide notement pour le Nuzlocke sur Pokemon Rubis.

Globalement plus qu'un guide ca sera une app avec des infos sur les pokemons, dresseurs, routes et cie.
On y retrouvera les information les plus importantes comme les base-stats, les capacités les taux d'apparition et autres.

Je cherche absolutment a recreer en n-ieme nuzlocke Tracker au contraire je cherche a creer un complement pour eviter de passer des jours sur le net a passer de wiki en wiki pour trouver les pokemons des dresseurs ou autres.
